  # Installation
  The following dependencies are needed to be installed to run the application: 
  * bcrypt 
  * sequelize
  * dotenv
  * express
  * express-handlebars
  * express-session
  * mysql2
* to install this make sure you run `npm i` and set a `.env` file with your own mysql creditentials
